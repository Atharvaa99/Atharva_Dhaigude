# Atharva Dhaigude — Developer Portfolio

Personal portfolio website built with React + Vite. Terminal-aesthetic, dark theme, fully responsive.

---

## Stack

- React 18 + Vite
- Plain CSS (no UI library)
- Google Fonts — Bebas Neue, Space Mono, DM Sans
- Deployed on Vercel

---

## Features

- Typewriter role animation in hero
- Custom cursor with glowing dot + lagging ring (desktop only)
- Per-project accent colors matching each project's own design system
- Live clock in navbar
- Mobile responsive
- Noise texture + grid background

## Run Locally

```bash
git clone https://github.com/Atharvaa99/portfolio
cd portfolio
npm install
npm run dev
```

---

## Deploy on Vercel

```bash
npm run build
```

Or connect the GitHub repo directly to Vercel — it auto-detects Vite.
